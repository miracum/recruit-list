# FHIR-based Clinical Trials Registry

## Project setup

```sh
npm install
# starts a FHIR-server preloaded with sample studies @ localhost:8089
docker-compose -f deploy/docker-compose.dev.yml
```

### Compiles and hot-reloads for development

npm run serve
